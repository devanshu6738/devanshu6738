### 📌 Featured Work

#### 🎥 [Computer Vision Notebook Collection](https://github.com/devanshu6738/Computer-Vision)
A hands-on collection of Jupyter notebooks covering the CV fundamentals — OpenCV basics, YOLOv8 object detection, Haar Cascade face detection, contour detection, convolution operations, and feature/video processing.

#### 🐆 Multi-Object Detection & Counting (Hoofed Animals Dataset)
Progressive build from traditional CV methods → HOG+SVM → custom CNN (TensorFlow/Keras) → a full from-scratch **R-CNN pipeline**: selective search, IoU-based training, bounding box regression, and Non-Max Suppression.

#### 🚗 Self-Driving Car — CARLA Simulator *(in progress)*
End-to-end autonomous driving stack: perception (YOLOv8/R-CNN), localization with optional Kalman filtering, A* + Frenet-frame motion planning, and PID-based control, all integrated in CARLA.
